Beijing Air Quality Prediction

This machine learning project aimed to predict Beijing's Air Quality Index (AQI) based on historical environmental and pollutant data. Using data from the UCI repository (2010–2014), we built two ML models: Ridge Regression and Random Forest Regressor.

What I Did: I performed end-to-end data preprocessing (handling missing values, Min-Max scaling), feature selection (PM2.5, temperature, dew point, wind speed, etc.), model training, and evaluation. I implemented both algorithms and visualized correlations between features and AQI.

What I Learned: This project deepened my understanding of supervised regression techniques, error metrics (MSE), model selection, and data visualization in environmental data analysis. I also learned to evaluate trade-offs between model complexity and generalization.

Optimization of CPFEM Parameters using ML

This Computational Engineering research project focused on improving the calibration process of Crystal Plasticity Finite Element Method (CPFEM) using machine learning. We integrated nanoindentation experiments, computational simulation (Abaqus), and neural networks to predict CPFEM parameters for advanced steels like DP1000 and QP1200.

What I Did: I developed and evaluated a machine learning pipeline, mainly using artificial neural networks (ANNs), to automate parameter estimation. I also contributed to simulation design and analysis in Abaqus.

What I Learned: This project gave me hands-on experience combining ML and materials science. I learned how to build physics-informed ML models, interpret force-displacement curves, and validate model outputs against experimental data.
